2、安装MySQL数据库

遇到you need to be root to perform this command 执行su并输入root密码解决

3、启动mysql服务

CentOS7 使用 systemctl

4、查看是否启动成功

显示active(running)表示启动成功

5、登录MySQL

  1. MySQL5.7.6 之后会在启动 mysql 进程的时候生成一个用户密码，首次登陆需要这个密码才行。密码保存在 mysql 进程的日志里，即(/var/log/mysqld.log)

  2. 不执行第四步在创建用户的时候会报

    You must reset your password using ALTER USER statement before executing this statement

6、创建新用户

创建mysql用户参考地址

为了和jdocs保持一致 username: vfly 密码: vfly

说明

  • username：你将创建的用户名
  • host：指定该用户在哪个主机上可以登陆，如果是本地用户可用localhost，如果想让该用户可以从任意远程主机登陆，可以使用通配符%
  • password：该用户的登陆密码，密码可以为空，如果为空则该用户可以不需要密码登陆服务器
